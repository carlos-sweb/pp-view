# pp-view
## Getting Started

In the web project include pp-view with:

```html
<script src="https://cdn.jsdelivr.net/npm/pp-view@latest/pp-view.min.js" ></script>
```

Or

## Install

```console
npm i pp-view --save
```
## Initialize

```javascript
var config = {}

var view = new ppView( config )
```
